# Hello, Blog Framework

Welcome to my blog.

The point of this version is not to fill everything in at once. The goal is to build a clear and durable framework first, then keep adding writing over time.

This is where I plan to collect project breakdowns, technical notes, devlogs, and longer-term thoughts.

## What this framework is trying to do

- Make the homepage feel like a personal entry point, not only a list of posts
- Separate About, Projects, and Writing into clear pages
- Support both Chinese and English
- Keep article bodies in Markdown files

## How I want to update it later

When I write a new post, the workflow should be simple:

1. add a new Markdown file
2. add metadata in `posts.json`
3. push to GitHub and let Render redeploy

That means the page code should stay mostly stable over time.
