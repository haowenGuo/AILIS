# Hello, Blog Framework

欢迎来到我的博客。

这个版本的重点不是一次把内容全写满，而是先把一个结构清晰、可长期维护的博客框架搭起来。

后续我会在这个框架上继续填项目复盘、技术笔记、开发日志和更长期的思考。

## 这个框架的目标

- 首页像个人入口，而不是单纯的文章列表
- About、Projects、Writing 分页清晰
- 支持中文和英文两个版本
- 文章正文改成 Markdown 文件驱动

## 后续如何更新

以后写文章时，我只需要：

1. 新增一篇 Markdown 文件
2. 在 `posts.json` 里补充元数据
3. 推到 GitHub，让 Render 自动部署

这样博客页面代码基本不需要频繁改动。
