## 从吉里吉里2迁移时，兼容性不是默认成立

README 最有工程价值的部分，是它明确提醒吉里吉里Z并不与吉里吉里2完全兼容。迁移旧 TJS2 脚本时，需要关注几个具体变化。

第一，默认文字编码已经改为 UTF-8。如果旧脚本仍按 Shift_JIS 编写，运行时需要通过命令行参数指定读取编码。第二，原本内置的 KAGParser 和菜单能力被插件化，需要时要链接对应 DLL。第三，在支持多点触控的设备上，触摸事件会被启用，可能从过去的鼠标处理变成触摸处理；不需要触摸时要显式关闭。第四，`PassThroughDrawDevice` 和其他被删除的方法需要改写。

这些提醒很适合作为迁移清单。它们不是抽象的“可能不兼容”，而是编码、插件、输入事件和绘制设备这些具体工程面。
