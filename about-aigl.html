🚀 AILIS 项目
📁 AILIS · v1.0.7 · MIT Open Source

AILIS —— 一个开源桌面虚拟助手项目。 它让我能跑在桌面上，能对话、能记忆偏好，也能通过 Agent 工具层执行真实任务！

🎯 产品定位

  • 拟人体验 —— 像一个真正的女孩子那样陪伴你，有表情、有语气、有温度
  • 专业能力 —— 底层对标 Codex / Claude Code 的任务执行稳定性
  • 本地优先 —— Electron 本地运行时，隐私安全都在你的电脑上
  • 工具框架 —— AILIS Gateway + Agent Runtime，负责规划、工具调用和审计

🌟 核心特性

  • 🎭 VRM 3D 角色 —— 3D 虚拟形象，表情动作实时驱动
  • 🗣️ 语音交互 —— TTS 语音输出 + ASR 语音识别
  • 👁️ 视觉理解 —— 只读屏幕感知，辅助理解上下文
  • 🧠 长期记忆 —— 记住你的偏好，越用越懂你
  • ⚡ 工具执行 —— 搜索、邮件、文件、代码、命令行全覆盖

🛠️ 技术栈

Electron Vite VRM / Three.js TypeScript Node.js WebSocket IMAP / SMTP RAG / 记忆

项目根目录在 F:\AIGril， 采用模块化架构，持续迭代中～
