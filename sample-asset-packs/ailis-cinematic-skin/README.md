# AILIS Cinematic Wave Skin

This is a local sample skin pack used to validate the AILIS asset-pack runtime.

It intentionally does not include a VRM model. Activating it keeps the built-in AILIS model and switches the render profile to `ailis_cinematic_rim_toon`.

Character packs can use the same manifest format with `"type": "character_pack"` and an `assets.vrm` path that points to a VRM file inside the pack directory.
